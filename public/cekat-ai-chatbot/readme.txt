AI-Powered Customer Service Chatbot untuk WordPress. Integrasikan chatbot cerdas ke website Anda dalam hitungan menit.

== Description ==

Cekat AI Chatbot adalah plugin WordPress yang memungkinkan Anda menambahkan chatbot bertenaga AI ke website Anda. Chatbot ini dapat:

* Menjawab pertanyaan pelanggan secara otomatis 24/7
* Belajar dari knowledge base yang Anda definisikan
* Menggunakan berbagai model AI (GPT-4, Claude, dll)
* Mengumpulkan leads dari visitor website

**Fitur Utama:**

* ✅ Setup mudah - hanya perlu Widget ID
* ✅ Kustomisasi tampilan widget
* ✅ Exclude halaman tertentu
* ✅ Responsive di semua device
* ✅ Dashboard untuk training AI

**Cara Kerja:**

1. Daftar akun di [cekat.biz.id](https://cekat.biz.id)
2. Buat chatbot dan isi knowledge base
3. Copy Widget ID dari dashboard
4. Paste di plugin ini
5. Selesai! Chatbot aktif di website Anda

== Installation ==

1. Upload folder `cekat-ai-chatbot` ke direktori `/wp-content/plugins/`
2. Aktivasi plugin melalui menu 'Plugins' di WordPress
3. Buka menu 'Cekat AI' di sidebar admin
4. Masukkan Widget ID dari dashboard cekat.biz.id
5. Klik 'Test Connection' untuk verifikasi
6. Save Settings

== Frequently Asked Questions ==

= Apakah saya perlu akun Cekat.biz.id? =
Ya, Anda perlu mendaftar di cekat.biz.id untuk mendapatkan Widget ID dan mengatur chatbot.

= Apakah plugin ini gratis? =
Plugin WordPress ini gratis. Layanan Cekat.biz.id memiliki tier gratis dengan batasan tertentu dan tier berbayar untuk fitur lebih lengkap.

= Bagaimana cara melatih chatbot? =
Login ke dashboard cekat.biz.id, lalu isi Knowledge Base dengan informasi tentang bisnis Anda.

= Apakah bisa dikustomisasi? =
Ya, Anda dapat mengubah warna, posisi, greeting message, dan lainnya melalui dashboard Cekat.
